# Future Considerations & Roadmap

This document serves as a repository for feature ideas, architectural improvements, and strategic considerations that are not yet part of the active development cycle but are valuable for long-term growth.

## Feature Ideas

### AI-Assisted Evergreen Engine 
**Concept**: Use AI to analyze post content and suggest whether it should be marked as "Evergreen" (Timeless).
- **Rationale**: While the creator's intention is the ultimate source of truth, an AI assistant could identify "timeless" patterns (e.g., deep-dive tutorials, philosophical reflections) vs. "tactical" patterns (e.g., "Happy Friday", event announcements).
- **Implementation Note**: This should be a subtle suggestion (e.g., a "Mark as Evergreen?" toast or tooltip) rather than an automated toggle, preserving the creator's authority over the signal-to-noise ratio in the Resurface list.
